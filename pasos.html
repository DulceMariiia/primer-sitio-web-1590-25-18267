Pasos para crear el sitio web personal

Este sitio fue realizado como parte de una tarea del curso Introducción a los Sistemas de Cómputo. A continuación, se describen los pasos que seguí:

  • 1.- Creé una carpeta con mi número de carnet como nombre: Sitio Web 1590-25-18267.
  • 2.- Utilicé Visual Studio Code para crear los archivos HTML del sitio.
  • 3.- Diseñé la estructura principal usando <frameset> para dividir el menú y el contenido.
  • 4.- En el archivo menu.html coloqué enlaces hacia las diferentes secciones del sitio.
  • 5.- Agregué una hoja de vida en hoja_vida.html.
  • 6.- Creé una tabla de notas de los primeros parciales con las materias cursadas con en notas.html.
  • 7.- Coloqué imágenes aleatorias dentro de una carpeta llamada imagenes/ y las mostré en galeria.html.
  • 8.- Inserté un video de YouTube usando la etiqueta <iframe> en video.html.
  • 9.- Agregué un juego simple hecho con JavaScript en juego.html.
  • 10.- Subí todos los archivos a GitHub y habilité GitHub Pages para que el sitio esté disponible en línea.
  • Usé como parte de la URL mi número de carnet para identificar mi sitio.
